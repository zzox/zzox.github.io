About

Use the keyboard (or tap any bottom button on mobile) to bind your keys to the buttons before starting. Tap the button or key according to the closest note, and hit all before the time limit is up. Challenges can be created and sent to others via a URL, but you must complete the challenge first.

patterns is made in HTML5, CSS, and Javascript and source code is available here. It is covered under the MIT license, feel free to alter/mod it as you please. Reach out on twitter if you have any questions. This project uses the Lexend font.

I have other games available on itch and newgrounds.

Please be kind to your keys.
